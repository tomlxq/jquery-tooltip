= A simple tooltip plugin
:image-width: 500

Author: tomLuo

Email: 21429503@qq.com

This is a demonstration. include fancybox css, please take care!

Example:
[source]
----
$("[class=toolImg]").tooltip();
$("[title^=header\\=]").tooltip({type: "html"});
$("[class^=toolTxt]").tooltip({type: "tooltip"})
----
